## PART 1 — Deploy Backend to Render

### Step 1 — Push backend to GitHub
```bash
cd /Users/fj/way-out
git init
git add backend/
git commit -m "Initial backend"
# Create a repo on github.com first, then:
git remote add origin https://github.com/YOUR_USERNAME/wayyout-backend.git
git push -u origin main
```

### Step 2 — Create Render account & deploy
1. Go to https://render.com → Sign up (free)
2. Click **New** → **Blueprint**
3. Connect your GitHub repo
4. Render detects `render.yaml` automatically and creates:
   - Web service: `wayyout-backend`
   - PostgreSQL database: `wayyout-db`
5. Click **Apply**

### Step 3 — Set environment variables on Render
In your Render dashboard → `wayyout-backend` → **Environment**:

| Key                   | Value                              |
|-----------------------|------------------------------------|
| `OPENAI_API_KEY`      | Your OpenAI key (for AI chat)      |
| `CORS_ALLOWED_ORIGINS`| `https://wayyout-backend.onrender.com` |

> `SECRET_KEY` and `DATABASE_URL` are auto-generated by Render via `render.yaml`.

### Step 4 — Run migrations (one-time)
In Render → `wayyout-backend` → **Shell**:
```bash
python manage.py migrate
python manage.py createsuperuser
```

### Step 5 — Verify backend
Visit: `https://wayyout-backend.onrender.com/api/venues/` — should return JSON.

## PART 2 — Deploy iOS App to App Store

### Prerequisites
- **Apple Developer account**: https://developer.apple.com — $99/year
- **Expo account**: https://expo.dev — free

### Step 1 — Create Expo account & log in
```bash
cd /Users/fj/way-out/mobile
npx eas-cli login
# Enter your expo.dev email & password
```

### Step 2 — Initialize EAS project (gets real project ID)
```bash
npx eas-cli init
# This links to your Expo account and prints the real projectId
# Copy the projectId it gives you
```

### Step 3 — Update app.json
Open `/Users/fj/way-out/mobile/app.json` and replace:
- `YOUR_EAS_PROJECT_ID` → the projectId from Step 2
- `YOUR_EXPO_USERNAME` → your expo.dev username

### Step 4 — Create App Store Connect listing
1. Go to https://appstoreconnect.apple.com
2. Sign in with your Apple Developer account
3. Click **+** → **New App**
4. Fill in:
   - **Name**: WayyOut
   - **Primary language**: English
   - **Bundle ID**: `app.wayyout.mobile` (register it first at developer.apple.com/account)
   - **SKU**: `wayyout-001`
5. After creating, note the **Apple ID** number (10-digit number in the URL)

### Step 5 — Update eas.json
Open `/Users/fj/way-out/mobile/eas.json` and replace:
- `YOUR_APPLE_ID@email.com` → your Apple ID email
- `YOUR_APP_STORE_CONNECT_APP_ID` → the 10-digit Apple ID from Step 4
- `YOUR_APPLE_TEAM_ID` → your Apple Team ID (find it at developer.apple.com → Account → scroll to bottom)

### Step 6 — Configure credentials
```bash
npx eas-cli credentials
# Select: iOS → production
# EAS will generate distribution certificate and provisioning profile automatically
```

### Step 7 — Build the iOS app
```bash
npx eas-cli build --platform ios --profile production
# This uploads your code to Expo's build servers
# Takes ~15-20 minutes
# You'll get a link to download the .ipa
```

### Step 8 — Submit to App Store
```bash
npx eas-cli submit --platform ios --latest
# Uses credentials from eas.json
# Uploads to App Store Connect automatically
```

### Step 9 — Complete App Store listing
In App Store Connect:

**App Information:**
- Category: **Food & Drink** (primary), **Entertainment** (secondary)
- Privacy Policy URL: Add your privacy policy URL (see below)

**Pricing:** Free

**Version Information (1.0.0):**
- Description (copy this):
```
WayyOut — Discover the best venues near you.

Find restaurants, clubs, rooftops and more with live crowd data.
See what's busy, what's open right now, and book your spot instantly.

Features:
• Real-time venue discovery with AI recommendations
• Live busy levels and crowd updates
• Instant table & guest list reservations
• Stories from venues you follow
• Available in English and French
```
- Keywords: `venues,nightlife,restaurants,reservations,nightout,discover,food,club`
- Support URL: Your website or email
- What's New: `Initial release`

**Screenshots required:**
- iPhone 6.5" (1284×2778): At least 3 screenshots
- iPhone 5.5" (1242×2208): At least 3 screenshots
- (Use simulator or the screenshots generated by the project)

**Age Rating:** Fill questionnaire → select "No" for all mature content → results in **4+**

**Review Notes for Apple:**
```
Demo account (customer):
Email: testuser@wayout.app
Password: WayyOut2024!

The app requires location permission to show nearby venues.
Backend hosted at: https://wayyout-backend.onrender.com
```

### Step 10 — Submit for Review
Click **Submit for Review** in App Store Connect.
Apple review takes 1-3 days on average.
